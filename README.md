# PiaRobot

`!!! work in progress!!!`

## Build Instructions
### Hardware used:
* Raspberry Pi 2
* 16GB SD card
* Raspberry Pi Camera
* with extra long cable
* PCA9586 16-Channel Servo Driver
* 2x SG90 servo motors
* Arexx Robby RP5/RP6 robot chassis

### Software
* node.js (socket.io,johnny-five,http,express,onoff,flite2)
* python3
